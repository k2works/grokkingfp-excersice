# Grokking Functional Programming - Python Implementation

Python で学ぶ関数型プログラミングの実装例です。

## 必要環境

- Python 3.11+
- pip

## インストール

```bash
# 開発モードでインストール
pip install -e ".[dev]"
```

## テスト実行

```bash
pytest
```

## 使用ライブラリ

- [returns](https://returns.readthedocs.io/) - Maybe/Result/IO コンテナ
- [hypothesis](https://hypothesis.readthedocs.io/) - プロパティベーステスト
- [pytest](https://docs.pytest.org/) - テストフレームワーク
